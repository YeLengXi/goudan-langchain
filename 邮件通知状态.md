## 快速配置（3 分钟）

### 方式 1: 使用配置助手（推荐）

**双击桌面**: `配置 goudan 邮件通知.bat`

配置助手会：
1. 打开 QQ 邮箱登录页面
2. 引导你获取授权码
3. 自动配置到 `.env` 文件
4. 发送测试邮件

### 方式 2: 手动配置

#### 步骤 1: 获取授权码

1. 访问: https://mail.qq.com
2. 登录你的 QQ 邮箱
3. 点击顶部 **设置** → **账户**
4. 向下滚动找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**
5. 开启 **POP3/SMTP 服务** 或 **IMAP/SMTP 服务**
6. 点击 **生成授权码**
7. 按照提示发送短信
8. 复制生成的 **授权码**（16 位字符）

#### 步骤 2: 配置 goudan

编辑 `E:\goudan-langchain\.env` 文件，添加：

```env
EMAIL_SERVICE=qq
EMAIL_USER=77037708@qq.com
EMAIL_PASS=你的授权码
EMAIL_TO=77037708@qq.com
```

#### 步骤 3: 测试邮件

```bash
cd E:\goudan-langchain
node test-email.js
```

#### 步骤 4: 重启 goudan

```bash
# 停止当前 goudan
taskkill /F /IM node.exe

# 重新启动
双击桌面: 启动 goudan.bat
```

## 故障排除

### 问题 1: 收不到邮件

**检查清单**:
1. ✅ 授权码是否正确（不是登录密码）
2. ✅ SMTP 服务是否已开启
3. ✅ 网络连接是否正常
4. ✅ 检查垃圾邮件文件夹

**解决方法**:
- 重新生成授权码
- 检查 QQ 邮箱 SMTP 服务状态
- 查看日志: `E:\goudan-langchain\logs\goudan.log`

### 问题 2: 邮件发送失败

**常见错误**:
```
Invalid login: 535 Login fail
```

**原因**: 授权码错误或 SMTP 服务未开启

**解决**:
1. 重新获取授权码
2. 确保 SMTP 服务已开启
3. 检查 `.env` 文件配置

### 问题 3: 邮件延迟

**原因**: QQ 邮箱可能有发送频率限制

**解决**:
- 等待几分钟后重试
- 减少通知频率（代码已优化）
