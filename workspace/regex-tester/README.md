# Regex Tester

This tool is a regex tester and debugger. It allows you to test regex patterns, extract capture groups, perform replacements, and split text.

## Features

- Test regex patterns
- Display match results
- Extract capture groups
- Regex replacement
- Common regex libraries
- Error handling

## Usage

To test a regex pattern:

```bash
node tester.js "/\d+/" --text "hello 123 world"
```

To validate an email:

```bash
node tester.js --email "test@example.com"
```
