## JSON数据处理工具

本工具提供JSON数据的格式化、排序、过滤和合并功能。

### 功能

- 格式化JSON数据
- 排序JSON数据
- 过滤JSON数据
- 合并JSON数据

### 使用方法

- 格式化：
  node json-tools.js format input.json
- 排序：
  node json-tools.js sort input.json --key name
- 过滤：
  node json-tools.js filter input.json --condition "age > 18"

### 注意事项

- 确保输入的JSON格式正确
- 使用时请按照正确的语法格式输入参数
