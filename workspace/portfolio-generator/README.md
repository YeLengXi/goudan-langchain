# Portfolio Generator

A static website generator for developers to create personal portfolios.

## Features

- Read personal information from portfolio.json
- Support name, position, bio, skills, project list
- Support social links (GitHub, LinkedIn, Email)
- HTML template system
- Responsive design
- Modern UI
- One-click generation of complete website
- Support custom theme color
- Generate CSS and HTML
- Local preview server
- Real-time reload
- Port configuration

## Usage

1. Run 'npm install'
2. Run 'npm run build' to generate the website
3. Open the generated website folder in a browser
