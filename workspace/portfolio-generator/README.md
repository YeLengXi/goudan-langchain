# Portfolio Generator

This is a static website generator for creating personal portfolio websites.

## Features

- Read personal information from `portfolio.json`
- Support name, position, bio, skills, project list
- Support social links (GitHub, LinkedIn, Email)
- HTML template system
- Responsive design
- Modern UI
- One-click website generation
- Support custom theme color
- Generate CSS and HTML
- Local preview server
- Real-time reload
- Port configuration

## Usage

1. Create a `portfolio.json` file with your personal information.
2. Run `npm start` to generate the website.
3. Open the generated website in your browser to preview.
