## Portfolio Generator

This is a static website generator for creating personal portfolios. It allows you to easily create a professional-looking portfolio website by filling out a configuration file.

### Features

- Read personal information from a configuration file
- Support for name, position, bio, skills, and project list
- Support for social links (GitHub, LinkedIn, Email)
- HTML template system
- Responsive design
- Modern UI
- One-click website generation
- Support for custom theme color
- Generate CSS and HTML
- Local preview server
- Real-time reload
- Port configuration

### Usage

1. Create a configuration file named `portfolio.json` with your personal information.
2. Run `npm start` in the terminal to generate the website.
3. Open the generated website in your browser to preview.
