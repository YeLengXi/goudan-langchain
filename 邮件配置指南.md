# 📧 goudan 邮件通知配置指南

## 快速配置（3 分钟）

### 步骤 1: 获取 QQ 邮箱授权码

1. 登录 QQ 邮箱: https://mail.qq.com
2. 点击顶部 **设置** → **账户**
3. 向下滚动找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**
4. 开启 **POP3/SMTP 服务** 或 **IMAP/SMTP 服务**
5. 点击 **生成授权码**
6. 按照提示发送短信
7. 复制生成的 **授权码**（16 位字符）

### 步骤 2: 配置 goudan

编辑 `E:\goudan-langchain\.env` 文件，添加：

```env
# 邮件通知配置
EMAIL_SERVICE=qq
EMAIL_USER=77037708@qq.com
EMAIL_PASS=你的授权码
EMAIL_TO=77037708@qq.com
```

### 步骤 3: 重启 goudan

```bash
# 停止当前运行的 goudan
taskkill /F /IM node.exe

# 重新启动
双击桌面: 启动 goudan.bat
```

## 通知类型

goudan 会在以下情况发送邮件通知：

1. ✅ **任务完成** - 每个任务完成后
2. 📄 **文件创建** - 创建新文件时
3. 📝 **Git 提交** - 代码提交后
4. 💰 **收入更新** - 收入统计更新
5. ❌ **错误发生** - 遇到错误时
6. 🚀 **启动通知** - goudan 启动时

## 故障排除

### 收不到邮件？

1. **检查授权码**：确认使用的是授权码，不是登录密码
2. **检查垃圾箱**：邮件可能在垃圾邮件文件夹
3. **检查网络**：确保网络连接正常
4. **查看日志**：`E:\goudan-langchain\logs\goudan.log`

### 授权码获取失败？

- 确保QQ账号已绑定手机号
- 确保QQ账号开启了SMTP服务
- 尝试重新生成授权码

## 安全提示

⚠️ **重要**：
- 授权码不是登录密码
- 不要分享你的授权码
- 定期更换授权码
- `.env` 文件包含敏感信息，不要提交到公开仓库
